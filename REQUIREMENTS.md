# BudgetBuddy - Requirements

## System Requirements

- **Node.js**: v18+
- **npm**: v9+
- Modern browser (Chrome, Firefox, Safari, Edge)

## Dependencies

### Production

- react@^18.2.0
- react-dom@^18.2.0
- react-router-dom@^6.20.1
- chart.js@^4.4.1

### Development

- typescript@^5.3.3
- vite@^5.0.10
- @vitejs/plugin-react@^4.2.1
- @types/react@^18.2.43
- @types/react-dom@^18.2.17

## Installation

```bash
npm install
```
